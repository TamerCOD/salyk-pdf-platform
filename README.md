# salyk-pdf-platform

Веб-платформа для работы с формой **ZVN STI-222** и API ГНС КР (cabinet.salyk.kg).

## Возможности

1. **PDF-генератор** (`/`)
   - Скачать шаблон Excel (`data_template.xlsx`).
   - Загрузить заполненный Excel — получить ZIP с отдельными PDF на каждую строку.
   - Опционально — выбрать колонку группировки (например, `E — Наименование налогового органа`).
     Все строки с одинаковым значением в этой колонке склеиваются в один многостраничный PDF.
   - Режим «все варианты сразу» — отдельные PDF + склейка по каждой содержательной колонке B–U.

2. **ИНН-парсер** (`/inn`)
   - Список ИНН текстом или Excel-файлом.
   - Платформа запрашивает `cabinet.salyk.kg/TinCheck/GetTaxPayer` и возвращает Excel с наименованиями и кодом УГНС.

## Локальный запуск

```bash
pip install -r requirements.txt
python app.py
# http://localhost:5000
```

## Docker

```bash
docker build -t salyk-pdf-platform .
docker run --rm -p 8080:8080 salyk-pdf-platform
```

## Деплой

Railway: подхватывает `Dockerfile` автоматически. Хелсчек — `GET /health`.
Кириллический шрифт DejaVu Sans ставится в образ через apt.

## Структура

```
app.py            # Flask-приложение
pdf_filler.py     # генерация PDF по шаблону
inn_lookup.py     # запросы к API salyk.kg
assets/
  form.pdf            # шаблон ZVN STI-222
  data_template.xlsx  # шаблон Excel
templates/        # Jinja2 шаблоны
static/style.css  # стили
Dockerfile
railway.json
requirements.txt
```
