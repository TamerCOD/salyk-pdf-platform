cabinet.salyk.kg :: INN lookup

Введите список ИНН (через пробел, запятую или с новой строки) или загрузите Excel, где в первой колонке стоят ИНН. Платформа дернёт API ГНС КР /TinCheck/GetTaxPayer и вернёт Excel с наименованием налогоплательщика и кодом УГНС.

// тайминги и лимиты:
  • Между запросами пауза 0.4 сек — чтобы не словить бан от ГНС.
  • Лимит: 500 ИНН за один запрос. Больше — бейте на части.
  • На 100 ИНН уходит ~45 секунд, на 500 ~3.5 минуты. Не закрывайте вкладку, пока идёт обработка.
  • Если ИНН не найден или вернул HTTP-ошибку — строка попадёт в выгрузку со статусом не найдено / HTTP NNN и подсветится цветом.

Зачем это нужно

  1. У вас есть список ИНН без наименований — нужно подтянуть «ФИО / название организации» из ГНС.
  2. Перед массовой генерацией ZVN-222 вы хотите сверить, что все ИНН живые и корректные.
  3. Кодом УГНС (поле rayon) можно сразу заполнить колонку D в шаблоне PDF-генератора.
// важно:
  • Это публичный API ГНС, который используется на форме регистрации cabinet.salyk.kg/account/register. Никаких особых прав не нужно, но не злоупотребляйте — серверу не нравятся залпы.
  • Кэширования на нашей стороне нет — каждый запрос идёт на ГНС. Если запрашиваете один и тот же ИНН повторно — это всегда новый сетевой вызов.
